# Novelpia-custom
공식앱 로딩이 너무 느려서 만든 커스텀 앱
## speeedup
- 노벨피아 공식 앱의 느린 로딩속도를 일부 개선
## functions
- 앱 시작 시 자동으로 내 서재 열람
- 소설 화면을 길게 터치하면 url 복사
- 화면 상단의 바로가기 버튼을 통해 복사한 소설 링크 바로가기 기능 추가
