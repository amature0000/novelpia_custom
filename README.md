# 법률 및 면책 조항
프로젝트의 목적은 개인적인 학습 및 편의 목적을 위해 제작되었으며, 노벨피아 서비스 이용 경험을 바탕으로 한 개인적 요구를 충족하기 위함입니다.

이 프로젝트는 개인이 제작한 비공식 프로젝트이며, 노벨피아와는 어떠한 공식적인 관계도 없습니다.

해당 프로젝트는 노벨피아의 공개된 인터페이스 및 웹 환경을 기반으로 하여, 앱 형태로 구현한 프로젝트입니다.

이 프로젝트는 노벨피아의 [사이트 이용약관](https://novelpia.com/page/terms_of_use)을 위반하지 않도록 사용하는 것을 전제로 하며, 문제가 될 소지가 있다고 판단될 경우 비공개 전환 및 삭제될 수 있습니다.
